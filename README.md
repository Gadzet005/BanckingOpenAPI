# Запуск

## Backend
- `cd backend`
- Создать `.env` файл (см. `env.example`)
- `pip install -r requirements.txt`
- `python src/manage.py migrate`
- `python src/manage.py runserver`

## Frontend
- `cd frontend`
- Создать `.env` файл (см. `env.example`)
- `yarn install`
- `yarn start`
